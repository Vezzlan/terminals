## Terminals - Java Stream API Playground

This project is a small Java/Maven playground focused on testing and learning two Stream API areas:

- `Collectors` API (for grouping, partitioning, mapping, counting, and downstream collectors)
- `Gatherers` API (Java 24+ stream gather operations)

## Tech Stack

- Java: `25` (configured in `pom.xml`)
- Maven: standard Maven project (`pom.xml`)

## Run Tests

If Maven is installed locally, run:

```bash
mvn test
```
